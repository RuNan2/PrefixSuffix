# PrefixSuffix
Plugin to set prefixes, suffixes, and nicknames for chats

## Supported Platforms
### Spigot
- Release v2.0.0 +
- Minecraft 1.21 +

## Command
### Prefix[접두사]
- /setprefix {player} {prefix}

### Suffix[접미사]
- /setsuffix {player} {suffix}  

### NickName[닉네임]
- /setnickname {player} {nickname}
